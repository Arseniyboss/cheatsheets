# nvm cheatsheet

| Command                            | Description                                             |
| ---------------------------------- | ------------------------------------------------------- |
| nvm install node                   | installs the latest node version                        |
| nvm install-latest-npm             | installs the latest npm version                         |
| nvm install [version-number]       | installs the specified node version                     |
| nvm uninstall [version-number]     | uninstalls the specified node version                   |
| nvm use [version-number]           | use the specified node version                          |
| nvm alias default node             | use the latest node version by default                  |
| nvm alias default [version-number] | use the specified node version by default               |
| nvm run [version-number] [file]    | run the specified file using the specified node version |
| nvm ls                             | lists installed node versions                           |
